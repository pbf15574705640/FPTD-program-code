# FPTD
Source code of paper, titled "FPTD: Super Fast Privacy-Preserving and Reliable Truth Discovery for Crowdsensing".

The code was tested within a MacBook Air equipped with an Apple M3 chip.

## 1. The main function to execute FPTD is in the below file: 

```FPTD_open_source/src/main/java/fptd/Main.java``` 


**Note:** The topological structure of Truth Discovery circuit in file TDOfflineOptimal.java should be exactly the same as the one in file TDOnlineOptimal.java!
